# 考研学习导图工具

一款基于思维导图的考研学习工具，支持PDF智能导入、知识点思维导图、题库管理与自评练习。

## 功能使用说明

### 1. 板块管理

应用按学科板块组织学习内容，预设五大板块：

- **数学**（math）、**英语**（english）、**政治**（politics）、**408专业课**（cs）、**八股文**（bagu）

你可以通过侧边栏底部的"新建板块"按钮创建自定义板块。每个板块拥有独立的思维导图、题库和练习记录。板块之间数据完全隔离，点击侧边栏即可切换。

### 2. 思维导图

#### 手动编辑节点

- 点击思维导图中的节点，右侧弹出详情面板
- 在详情面板中可**编辑节点名称**、**添加子节点**、**删除节点**
- 右键点击节点弹出快捷菜单（移动端长按），支持添加子节点、编辑名称、删除

#### PDF智能导入

1. 点击顶部导航栏的"导入PDF"按钮
2. 选择或拖拽PDF文件到上传区域
3. 选择目标板块，点击"开始AI分析"
4. 系统自动提取前200页文本，匹配该板块预设关键词，生成结构化思维导图
5. 导入的节点会标记来源文件名和创建时间

**大文件提示**：文件超过200页时解析可能较慢，建议拆分后分批导入。

#### 节点详情

每个节点包含：

- **基本属性**：名称、来源文件、创建时间
- **高频考点**：可添加/删除考点标签，以橙色左边框列表展示
- **关联题目**：显示该节点下的所有题库题目，可直接查看答案
- **练习入口**：点击"开始练习"随机抽取3道关联题目进行自评

### 3. 题库管理

#### 添加题目

- 在节点详情面板点击"为该节点添加题目"
- 或在题库总览页面点击"+ 添加题目"
- 题目信息包括：绑定的知识点节点、题型（简答/选择/填空）、题干、答案要点

#### 题库总览

- 左侧板块列表显示各板块题目数量
- 支持按**知识点节点**筛选、按**题型**筛选、**关键词搜索**
- 支持**批量选择**和**批量删除**题目
- 支持**导出题库为JSON文件**备份
- 每道题可查看答案、编辑或单独进入练习模式

### 4. 练习模式

#### 节点练习

从思维导图节点进入，系统从该节点关联的题目中随机抽取**3道题**。逐题展示题干，点击"显示答案"后自行对比，然后自评"正确"或"错误"。完成全部题目后显示正确率。

#### 题库练习

从题库总览选择单道题目进入练习。带**计时器**功能，记录答题用时。同样支持答案揭示和自评。

### 5. 数据与同步

所有用户数据（板块、思维导图、题库、练习记录、主题偏好）仅保存在浏览器本地：

- 业务数据存储在 **IndexedDB**（通过 localforage）
- 主题和偏好存储在 **localStorage**
- **不会上传到任何远程服务器**
- **不会被 Git 版本管理系统收录**

建议定期使用导航栏的"导出"按钮备份数据，生成JSON文件保存到本地。

### 6. 界面与交互

- **深色/亮色双主题**：点击导航栏或底部导航栏的太阳/月亮图标切换
- **折叠侧边栏**：点击左上角汉堡菜单展开/收起板块列表（桌面端浮层，移动端全屏覆盖）
- **思维导图交互**：支持拖拽平移、滚轮缩放（0.3x-3x）、节点展开/折叠
- **滑出详情面板**：点击节点后从右侧滑入，关闭按钮或点击其他区域收起
- **移动端适配**：底部毛玻璃导航栏，详情面板从底部弹出
- **PWA支持**：可安装到桌面，离线使用

| 字体 | Calistoga + Inter + JetBrains Mono（Google Fonts） |

## 数据存储声明

**所有用户数据保存在浏览器本地**（IndexedDB / localStorage），不涉及任何远程服务器，不存在数据上传或云端同步行为。应用程序不包含任何网络请求（除加载字体和pdf.js worker CDN外）。

本项目 `.gitignore` 已配置忽略所有本地数据文件格式（`.localStorage`、`.ldb`、`*.log`、备份JSON文件等），确保用户数据不会随代码提交到Git仓库。
